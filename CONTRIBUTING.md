# Contributing to NeuroGuard

Thank you for your interest in contributing.

NeuroGuard is a privacy infrastructure project focused on protecting neural and biometric data.

## Development Setup

Clone the repository:

git clone https://github.com/yourusername/neuroguard-sdk

Install dependencies:

pip install -e ".[dev]"

Run tests:

pytest tests/

## Contribution Guidelines

- Write clear code
- Add tests for new features
- Follow existing project structure
- Keep privacy and security considerations in mind

## Pull Requests

Please open a pull request describing:

- what the change does
- why it is needed
- any relevant tests
